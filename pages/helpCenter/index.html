粮票相关说明

  1. 什么是粮票？

    【粮票】是吉粮惠民APP为每一位消费者提供的“补贴”，可以在APP内进行消费抵扣和补贴提取，1粮票=1元钱。

  2. 如何获取粮票？

    在吉粮惠民APP上消费即可获得粮票，最高可获得与商品价格等额的粮票补贴。例：在市场上购买某款商品需支付100元，而在吉粮惠民APP上购买同款商品，在支付100元后，还可获赠10%-100%的粮票补贴，即获得10-100粮票。

  3. 如何使用粮票？

    在商城【粮票区】、【聚划算】、【拼手气】等活动专区可以通过粮票购买商品，最高可100%抵扣。

    例：

    在市场上购买某款商品需支付100元，而在吉粮惠民APP购买同款商品（50%粮票抵扣）：支付50元现金+50粮票即可获得该商品；若为100%粮票抵扣，只需支付100粮票即可获得该商品。

    另外，粮票除了抵扣商品金额，还可兑换，通过APP点击“我的——粮本——兑换”进行兑换。

下单问题

无法下单可能有以下几种情况：

  1. 秒杀或拼手气类商品，您购买的数量可能超过限购的数量；

  2. 其他会员在您下单之前，成功拍完了剩余商品件数，您可以尝试刷新商品页面；

  3. 只能用粮票购买的商品，商品所需粮票数量超过您可使用的粮票数量；

  4. 海淘商品需要填写正确的姓名和身份证号，若信息错误或缺失则无法正常下单；

  5. 商品为VIP商品，需开通VIP会员服务才可购买；

  6. 商家已设置起送价，您购买的商品低于起送价；

  7. 为保障用户的账号安全，首次使用余额支付需设置支付密码；

  8. 使用余额支付时，支付密码输入错误。

*温馨提示

若以上情况都不是，可能是您的网络原因，建议您刷新订单页面或者更换设备提交。

物流情况

  1. 什么时候可查看订单物流信息？

    订单付款成功，商家发货之后您可以在订单列表中点击查看物流按钮。

  2. 物流信息没有更新。

    1. 看不到物流信息，有可能是物流公司收到物件还没有及时扫描；

    2. 物流分发信息长时间不更新，可能是物流目的地比较偏远，中间派送时间很长；或者处于快递高峰期；

    3. 物件已经分配派件员，长时间不更新，可能是物流地址与联系方式不正确，或者被他人代签后没有及时上传收件人信息等。

*温馨提示

由于海外商品需清关，物流时间稍有延迟。若长时间未更新物流，可联系物流公司查询具体情况。

售后服务

  1. 退换货规则

    1. 签收商品之日起七天内（按照签收后的第二天开始计算时间），商品完好可申请7天无理由退货。完好标准：能够保持原有品质、功能，商品本身、配件、商标标识齐全的，视为商品完好。

    2. 部分商品因商品属性问题，不支持7天无理由退货

    3. 退货时请保持主商品完好，附件齐全，并将商品的赠品一并返回；如赠品丢失，退款需扣除赠品费用。

    4. 海淘类产品客户其他原因退货，需额外承担10-100元的清关处理费用。

  2. 收货时间判定标准。

    按照第三方物流平台显示的实际到货日期第二日为准。如果第三方合作伙伴不能有效返回签收日期，则吉粮将根据距离等因素和客户确认实际到货第二日开始计算。

  3. 下述商品不支持无理由退货。

    1. 个人定制类商品；

    2. 生鲜类、鲜活易腐类商品；

    3. 在线下载或者您拆封的音像制品，计算机软件等数字化商品；

    4. 交付的报纸期刊类商品；

    5. 根据商品性质不适宜退货，并经您在购买时确认的商品：

      1. 拆封后易影响人身安全或者生命健康的商品，或者拆封后易导致商品品质发生改变的商品；

      2. 一经激活或者试用后价值贬损较大的商品；

      3. 销售时已明示的临近保质期的商品、有瑕疵的商品；

      4. 其他根据商品性质不适宜退货，在商品页面标注“不支持无理由退货”类的产品

    6. 无法保证退回商品完好的商品（能够保持原有品质、功能，商品本身、配件、商标标识齐全的，视为商品完好）。

  4. 以下情况不予办理退换货

    1. 任何非吉粮惠民自营商城及联盟店出售的商品（序列号不符）；

    2. 过保商品（超过三包保修期的商品）；

    3. 未经授权的维修、误用、碰撞、疏忽、滥用、进液、事故、改动、不正确的安装所造成的商品质量问题，或撕毁、涂改标贴、机器序号、防伪标记；

    4. 包凭证信息与商品不符及被涂改的商品；

    5. 其他依法不应办理退换货的商品。

  5. 生鲜类产品售后标准：

    1. 生鲜类商品（蔬菜水果、水产海鲜、肉禽蛋奶等）由于受季节、气候、生长环境等因素影响，不同批次略有差异属正常现象；

    2. 图片及信息仅供参照，商品将以实物为准。因拍摄灯光及不同显示器色差等问题可能造成商品图片与实物有一定色差，不属质量问题；

    3. 为保障您的权益，请您签收时与配送人员当面核对商品的种类、数量、规格、赠品、金额、保质期等是否正确，请在确认无误后再进行签收。如有问题请及时向配送人员反馈，一旦签收，吉粮惠民商城将不受理此类退货、补货等要求；

    4. 因客户使用或保管不当导致出现质量问题、变质的商品,无法办理退换货；

    5. 生鲜类产品如出现破损或腐坏的，可以要求在面单上注明，必须先行签收，不可以拒签，在签收后48小时内反馈给客服人员。吉粮惠民商城会根据损失情况，赔偿或补发。

  6. 美妆类、个护类产品售后注意事项：

    1. 出于安全考虑，贴身用品、美妆类非质量问题不支持退换货。国家法律规定有质量问题的除外；

    2. 产品过敏类不适用于质量问题；

    3. 包装含防伪标签类产品，刮开或破损后不支持退换货；

    4. 因个人原因造成的商品损坏，不予退换货；

    5. 礼包、套装类商品不接受针对部分商品的退换货申请；

  7. 母婴类产品产品售后注意事项：

    1. 母婴食品、婴儿用品、贴身衣物属于特殊用品，一经售出，不支持退换货

    2. 产品过敏类不适用于质量问题；

    3. 因个人原因造成的商品损坏，不予退换货；

    4. 包装含防伪码的商品防伪码一经刮开不支持7天无理由退货；

    5. 礼包、套装类商品不接受针对部分商品的退换货申请；

  8. 食品类、保健品产品售后注意事项：

    1. 食品类、饮料、保健品类特殊商品，一经售出，不予退换货；

    2. 包装含防伪码的商品防伪码一经刮开不支持7天无理由退货；

    3. 对于保健类商品，请你食用前仔细阅读书明说内家，禁忌人群，请勿食用。

    4. 礼包、套装类商品不接受针对部分商品的退换货申请；

  9. 服饰鞋靴类产品售后注意事项：

    1. 因客户个人原因造成的商品损坏（如自行修改尺寸，洗涤，皮具打油，刺绣，长时间穿着、体味及香水污染等），无法退换货；

    2. 如遇过敏问题产生退货需要提供医院的相关证明；

    3. 礼包、套装类商品不接受针对部分商品的退换货申请；

    4. 防伪标识码刮开或撕损、商标标识、吊牌撕毁、唯一性三包卡缺失或撕损，不支持7天无理由退货；

    5. 出于安全和卫生考虑，贴身用品如内衣裤、袜子、泳衣等商品不予退换货。

  10. 钟表首饰类产品售后注意事项：

    1. 如有缺失附件不支持7天无理由退货；

    2. 佩戴饰品时不慎造成变色、斑点、污染、折断、变形、开裂、松动、掉石、出现擦痕等不予退换货；

    3. 未经授权的修理、误用、疏忽、滥用、事故、改动、不正确的安装，保存不当；

    4. 商品使用中出现的磨损；

    5. 黄金产品除质量问题以外，不予退换货；

    6. 包装含防伪码的商品防伪码一经刮开不支持7天无理由退货；

    7. 对于高档首饰都附带国家级宝玉石鉴定中心出具的鉴定证书，如果您对此有任何质疑，请到出具该证书的鉴定机构进行检测；

    8. 手表只保修机芯，如非人为原因造成损坏可按照流程办理退换货；

    9. 贵金属、珠宝及其镶嵌制品中的黄金、铂金、钯制品及K金含量不足发票或饰品签上标注的（允许误差范围+0.01g~-0.01g内的）；

    10. 手表表带经过调整，不支持无理由退换货，但有质量问题除外；

*温馨提示

在商品退货时，需扣除购买该商品时得到的粮票和提现提取的现金。

收不到验证码

收不到验证码可能有以下几种情况：

  1. 每个手机号每天只能获取五次验证码，如果今日已超过次数请明天再尝试。

  2. 已安装安全软件的智能手机用户验证码被拦截。建议暂时关闭拦截功能，然后再次尝试获取验证码。

  3. 短信网关拥堵或出现异常时会导致已发送的短信出现延时或丢失的情况，建议过段时间再尝试获取。

  4. 受手机服务运营商业务影响，手机在境外使用或使用境外手机都无法接收验证码，建议更换手机尝试。

  5. 其他手机号码可以收到验证码，但自己始终收不到，可能是手机本身的原因，建议将手机卡换到别人的手机上再次进行尝试。

*温馨提示

如果您遇到页面提示“验证码错误”，建议您确认验证码输入是否有误，如果获取了多条验证码，请输入最后收到的验证码。
